# CoinShop-Plugin
Minecraft Coin Shop Plugin that uses the Coin API to buy and sell itens between players.

Soft depends on vault.

Depends on CoinCard Plugin:

https://www.spigotmc.org/resources/coin-card.129770/


API 1.20+

Coin API> https://bank.foxsrv.net/

You can host your own: https://github.com/FoxUshiha/DC-Coin-Bot


Commands:


/cshop or /coinshop

/cshop name - renames your store

/cshop cancel - cancel items

/cshop open - opens a specific player shop

/cshop reload - reloads configurations (default OP)

/cshop sell amount price (in bitcoin like: 0.00000001)


Permissions:


coinshop.use (users)

coinshop.admin (use reload command)

coinshop.* (all)
